A professional, fully responsive Student Portfolio website built from scratch using HTML5, CSS3, and Vanilla JavaScript. The objective of this project is to showcase a student's achievements, academic performance, club activities, competitions, and personal growth in a visually appealing and professional manner.

## ✨ Key Features

- **Modern UI/UX**: Built with contemporary design principles, glassmorphism elements, and vibrant gradients.
- **Dark & Light Mode**: Integrated theme toggle that seamlessly switches color palettes and saves user preferences.
- **Scroll Animations**: Smooth, performance-optimized reveal animations using the Intersection Observer API.
- **Fully Responsive**: Adapts perfectly to mobile phones, tablets, and desktop displays.
- **Zero Dependencies**: The frontend relies purely on Vanilla JavaScript and custom CSS (no heavy frameworks like jQuery, Bootstrap, or Tailwind).

## 🗂️ Project Structure

The project is organized into clean, modular files for easy maintenance:

- `index.html` - The main portfolio page containing all sections.
- `/css/` - A modular CSS architecture:
  - `design-tokens.css`: Core CSS variables (colors, typography, spacing).
  - `base.css`: CSS resets and foundational HTML styling.
  - `components.css`: Reusable UI elements (buttons, cards, badges, navbars).
  - `sections.css`: Specific layout styles for the Hero, About, Gallery, etc.
  - `animations.css`: Keyframe definitions for entrance and hover effects.
  - `dark-mode.css`: Color variable overrides for the dark theme.
- `/js/`
  - `main.js`: Handles the mobile menu, dark mode toggling, scroll animations, and typing effects.
- `/assets/` - Contains downloadable files like the student CV.

## 🚀 How to Run Locally

Because this is a purely static website, running it is incredibly simple:

1. Clone or download this repository.
2. Open the folder on your computer.
3. Double-click `index.html` to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
*Note: No local server, Node.js, or build tools are required!*

## 🏆 Purpose

This project was designed and developed to highlight the journey of a student in the GEMA Clubs. It effectively presents everything from Abacus Olympiad medals and Science fair projects to Scratch coding games and public speaking achievements in one cohesive, interactive experience.
